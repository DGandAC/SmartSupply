# SmartSupply

SmartSupply to projekt wspierający analizę sprzedaży, zapasów, prognozowania zapotrzebowania i rekomendacji zakupowych.

## Struktura

```text
backend/
  python/
    api/        # FastAPI dla aplikacji mobilnej
    SmartSupply_Dashboard.py
database/
docs/
mobile/
  android/     # aplikacja Android
```
